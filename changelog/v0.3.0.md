# v0.3.0 — 2026-04-24

_Since [v0.2.1] (2026-04-24)._

## Features

- **Rich text build-step editor** — Tiptap-powered editor in the Instructions tab, with inline links, bullet/numbered lists, and a placeholder. Replaces the previous plain-textarea workflow.
- **Grain-lock safety check** — locking a part's grain now runs a fit check against matching stock boards; if the lock would make the part unplaceable, a `GrainLockConfirmModal` asks for explicit confirmation before applying it.
- **Leftover list in layout view** — unplaced parts and off-cuts now render in a dedicated `LayoutLeftoverList` panel, separated from the main board list for quicker scanning.

## Code Quality

- **Layout tab decomposition** — split `LayoutList.vue` into focused pieces (`LayoutLeftoverList`, `useGrainLockConfirm`, shared `layout-types`), shedding ~90 net lines from the main component.
- **`canFitOnAnyBoard` utility** — extracted the part-vs-board fit check into a pure helper with full unit test coverage.
- **Instructions tab slimming** — extracted the editor into `RichTextEditor.vue` and trimmed ~200 lines of inline editor plumbing from `InstructionsTab.vue`.
